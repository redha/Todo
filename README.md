# todo 

## Download the Project
```
git clone https://github.com/redha/Todo.git
```

## Project setup
```
cd todo
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
